УСТАНОВЛЕННЫЕ ПАКЕТЫ:
psycopg2, bs4, typer


КРАТКОЕ ОПИСАНИЕ ФАЙЛОВ:
parser.py -
    скрипт с функциями для получения информации с сайта ВШЭ
db_manager.py -
    скрипт с функциями для создания таблиц, заполнения их данными, полученными с сайта
classes.py -
    содержит классы для упрощения сохранения информации и более удобной работы с ней
db_config.py -
    файл с конфигурациями, необходимыми для подключения к базе данных
parsing_sources.txt -
    файл с перечислением веб страниц и кратким описанием полезной информации, расположенной на них
main.py -
    скрипт с функциями и командами для получения данных из бд через командную строку:
Backup.cmd -
    скрипт для создания бэкапа базы данных
HSE_dump.zip -
    архив с файлом Restore.cmd
Restore.cmd -
    скрипт для восстановления базы данных


КОМАНДЫ ДЛЯ КОМАНДНОЙ СТРОКИ:
show-faculties:
    names, links - параметры команды (указываются в любом порядке через дефис (-))

show-bachelor-courses:
    names, links, durations, languages, campuses, places, educationforms (указываются в любом порядке через дефис (-))

show-magister-courses:
    names, links, durations, languages, campuses, places, educationforms (указываются в любом порядке через дефис (-))

Например, команда - main.py show-bachelor-courses names-links-durations
выведет три столбца данных по курсам бакалавриата: названия, ссылки и продолжительность обучения каждого курса


БАЗА ДАННЫХ:
1) Разархивировать HSE_dump.zip, в котором лежит файл HSE.dump
2) Команда: pg_restore -U username -d dbname -1 filename.dump
